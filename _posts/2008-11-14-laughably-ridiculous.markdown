<p>Okay, although I'm an open-source devotee, I've actually intellectually bantered in favor of copyright law. I know, I'm sorry; but I can understand the philosophical underpinning of *a* copyright scheme ... even if I don't agree with its effectiveness.</p>
<p>But <a href="http://torrentfreak.com/record-labels-to-sue-vuze-limewire-and-sourceforge-081114/">this</a> is just getting absurd.</p>
<blockquote><p>Interestingly, SPFF is also going after Sourceforge, the open source development website, because it hosts the P2P application Shareaza.</p></blockquote>
<p>So let me state this matter-of-factly:</p>
<p>In suing SourceForge, SPFF is not suing an entity who distributes copyrighted material. They're not even suing someone who develops software that <span style="font-weight:bold;">might</span> be used to distribute copyrighted material. SPFF is suing <span style="font-style:italic;">someone (i.e., SourceForge) who develops software (i.e., <a href="http://sf.net/">sf.net</a>) that <span style="font-weight:bold;">might</span> be used to develop or distribute software (i.e., <a href="http://shareaza.sf.net/">Shareaza</a>) that <span style="font-weight:bold;">might</span> be used to distributed copyrighted material</span>.</p>
